# AI Resume Analyzer

## Project Overview
AI Resume Analyzer is an offline desktop application developed using Python. The system analyzes resumes, extracts skills, and matches them with predefined job roles using AI-inspired keyword matching techniques.

## Features
- Resume PDF Upload
- Skill Extraction
- Compatibility Score Analysis
- Missing Skill Detection
- Report History
- Dark Mode GUI
- Offline Functionality

## Technologies Used
- Python
- CustomTkinter
- PyPDF2
- JSON
- VS Code

## Installation

Install required libraries:

pip install customtkinter PyPDF2 tkinterdnd2

## Run Project

python Main.py
